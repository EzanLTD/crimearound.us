What’s happening around you?

This app provides a unique way of finding out.

Draw a map…

Using the tools on the left side of the map interface, you can draw a custom shape on the map to define the area that you are interested in. When you’re done drawing, the area you drew will immediately be populated by dots representing the various reported crimes that took place between 7 and 14 days ago. To customize the date range or limit the types of crimes returned, you can use the controls across the bottom of the map. Both date fields expect a date formatted like so YYYY-MM-DD

…get a report

Once you’ve honed in on what it is that you want to look at, you can click on “Get a Report” which will generate an Excel Spreadsheet of everything you’re looking at. Have a use for other formats? You can submit an issue over here and tell me what you’re after.

How does this thing work, anyways?

In a nutshell, the portion of the app that you are interacting with here and on the map page is being hosted entirely out of an S3 bucket. I’ve got a MongoDB instance that has all crime reported in Chicago from 2001 until a week ago. In front of the database is a very simple Flask app that gives this app a way to talk to it by responding with JSONP. Because it uses JSONP, the database can be used by client-side javascript from anywhere on the web. Or, with a bit of trickery, from server-side apps as well. Want to use this data? Read the docs…
