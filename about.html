How does this thing work, anyways?

In a nutshell, the portion of the app that you are interacting with here and on the map page is being hosted entirely using Github pages. There is a MongoDB instance that has all crime reported in Chicago from 2001 until a week ago. In front of the database is a very simple Flask app that gives this site a way to talk to it by responding with JSONP. Because it uses JSONP, the database can be used by client-side javascript from anywhere on the web. Or, with a bit of trickery, from server-side apps as well. Want to use this data? Read the docs…
